### Week 2 Achievements:

* Continue watching the designated YouTube tutorial videos for the week.
* **For Lab 000004 (Amazon EC2):**
    * Learned the basic features of the core AWS virtual server service.
    * Learned how to select Instance Types and manage AMIs, Backups, and Key Pairs.
* **For Lab 000005 (Amazon RDS):**
    * Understood how to deploy a relational database for structured data storage requirements (OLTP).
    * Learned how to configure high availability with Multi-AZ deployments and scale read performance using Read Replicas.
* **For Lab 000006 (ASG & ELB):**
    * Practiced deploying the FCJ Management application.
    * Got familiar with using Amazon EC2 Auto Scaling Groups for flexible scaling.
    * Learned about Elastic Load Balancing to distribute traffic to the Application Tier.
* **For Lab 000007 (AWS Budget):**
    * Explored the 4 types of budgets (Cost, Usage, RI, and Savings Plans Budgets) to monitor costs within the AWS account.
* **For Lab 000008 (AWS CloudWatch):**
    * Learned how to collect performance data via CloudWatch Metrics and Logs from applications running on EC2 instances.
    * Practiced setting up Alarms and building simple Dashboards for resource monitoring.
* **For Lab 000010 (Route 53 Resolver):**
    * Understood the concept of a hybrid DNS architecture.
    * Got familiar with utilizing Outbound Endpoints, Inbound Endpoints, and Resolver Rules to integrate an on-premise DNS with Amazon Route 53.
