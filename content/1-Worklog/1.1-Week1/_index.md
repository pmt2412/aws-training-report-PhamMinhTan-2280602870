### Week 1 Achievements:

* Attended the kickoff meeting, understood the rules, and grasped the implementation roadmap of the program.
* Read and mastered the program's general rules, including: regulations on attendance and roll call, time, dress code and demeanor, office access, eating and using office supplies, and restroom usage.
* Understood the process and requirements for registering to work offline at the 26th Floor, Bitexco Financial Tower (2 Hai Trieu St., Ben Nghe Ward, Dist. 1), as well as the standard code of conduct in the group chat.

* Successfully created an AWS account and completed 5 practical tasks to receive an additional 100 AWS credits.
* Finished watching the YouTube tutorial playlist to build foundational knowledge of the AWS ecosystem.
* **For Lab 000002 (Access Management with IAM):**
  * Understood the centralized Authentication and Authorization mechanisms of AWS.
  * Mastered and practiced creating the 4 core components: IAM Users, IAM Groups, IAM Policies, and IAM Roles.
  * Grasped security authorization methods adhering to the principle of least privilege.
* **For Lab 000003 (Amazon VPC and AWS Site-to-Site VPN):**
  * Successfully deployed an Amazon VPC virtual network architecture adhering to the AWS Well-Architected Framework standards.
  * Set up and configured critical network security layers using Security Groups and Network ACLs.
  * Successfully configured an EC2 Instance environment and established a secure AWS Site-to-Site VPN connection between a simulated on-premise environment and AWS Cloud.
